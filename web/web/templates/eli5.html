振興抵用券是什麼？

What

因武漢肺炎 ( 新型冠狀病毒 ) 帶來的經濟影響，經濟部與文化部推23億元振興抵用券，結合國民旅遊來刺激消費。振興抵用券隨住宿發放，每房新台幣600元到800元之間。振興抵用券適用於受衝擊服務型產業，包含餐飲、商圈、夜市及藝文等四類產業，全台共約35萬店家適用。

使用期限

When

推出時間：待疫情穩定後半年至一年內發放。

使用期限：目前仍未定案，但預計會開放使用到2021年6月底。

發放對象

Who
本國國人，不分年齡限制，入住全台符合旅遊補助飯店名單的飯店後，由旅宿業者提供振興抵用券800元。 ( 尚不確定透過訂房網是 否能領取抵用券 )

如何領取

How

領取方式：全台不限住宿地區使用，只要入住符合旅遊補助的國內觀光飯店，由旅宿業者提供。

抵用券額度：每房補助 800 元，全國的商圈、夜市、餐廳、藝文產業，4種產業各有200元的消費額度。

抵用券形式：將採電子與紙本票券方式並行，紙本票券單張面額 50 元，電子票券將研擬使用App。

哪裡可用

Where
針對受疫情衝擊較大的服務業，包含商圈 ( 約28萬家 ) 、夜市 ( 約1萬家 ) 及全國餐飲業 ( 14.5萬家 ) 、藝文產業 ( 1700 家 ) ，扣除重複店家大約35萬家皆可使用。
