振興三倍券是什麼？

What

為了刺激消費，行政院推出振興三倍券，讓民眾可以用一千元買到一本三千元的現金券 ( $200*5，$500*4 ) 。用來刺激國內經濟活絡的紓困補助措施，凡國民人人都可領取，並在消費時替代現金使用。官方資訊可查看振興三倍券官網

使用期限

When

購買時間：109/7/1 開始預購或綁定。

使用期限：109/7/15 開始至 109/12/31 前必須使用完。

發放對象

Who

本國國人：不分年齡限制 ( 今年出生的新生兒到100歲以上都可使用 ) 。

非本國籍配偶：有居留證的陸配、外籍配偶也可領取。

中低收入、弱勢：全國 120 萬的中低收入者及弱勢民眾，政府將直接匯入 1000 元至戶頭中，並於摘要處註明「換三倍券」，使他們得以兌換振興三倍券進行消費。

如何領取

How

[ 實體紙本 ]

實體券有 200 元券 5 張和 500 元券 4 張，等同現金、不找零、不可用於儲值。( 原則上不能找零，若攤商願意找零，方便消費者購買，不會禁止，但是消費者不能強迫攤商找零。 )

超商領取

  • 7/1 起，持健保卡至四大超商或到振興券網站插卡預購並繳費 ( 比照口罩實名購買流程 )。

  • 7/15 起，於四大超商 ( 7-11、全家、萊爾富、OK ) 領取 3000 元振興三倍券。預計 8/1 ~ 8/8 開始第二階段的預購。

郵局領取

7/15 起，帶健保卡及身分證/居留證及 1000 元，至郵局領取振興三倍券。

[ 數位綁定 ]

皆於 7/1 開始，綁定好支付系統，7/15 開始累積消費。累積消費滿 3000 元後，回饋 2000 元。如果欲轉換已綁定的數位振興三倍券，可以於取消原方案後，綁定其他支付方式，但累計消費金額 ( 須達 3000 元 ) 將重新計算。

行動支付

  • 7/1 起，綁定任一行動支付 ( LINE PAY、街口支付、Pi支付、台灣PAY )

  • 7/15 開始累積消費，累積消費金額滿 3000 元後，需到四大超商透過超商靠卡申請。申請通過後，便可領取 2000 元的振興三倍券，將直接匯入 2000 元至電子支付帳號中。

電子票證

  • 7/1 起綁定任一電子票證 ( 悠遊卡、一卡通、icash 與 HappyCash 有錢卡 )

  • 7/15 開始累積消費，累積消費金額滿 3000 元後，將會收到 APP 通知。收到 APP 通知後便可獲得 2000 元的振興三倍券，須至四大超商靠卡回存 2000 元至該電子票證中。

信用卡

  • 7/1 起綁定任一信用卡 ( 於該銀行官網、APP登錄區進行綁定。

  • 7/15 開始累積消費，累積消費金額滿 3000 元後，將會收到簡訊通知。收到簡訊通知後便可獲得 2000 元的振興三倍券，將直接折扣 2000 元於隔月該信用卡帳單中。

哪裡可用

Where

可以使用：實體店面、餐飲、夜市、傳統市場、商圈、藝文活動、旅遊住宿、高鐵與台鐵等，吃、喝、玩、樂、買均可使用。

無法使用：電商網購、繳稅、罰單、規費、菸品、保單、股票、國民年金、信用卡費、禮券、儲值等11類無法使用三倍券。

其他：出售票券、提供實體經驗等電商可以適用，例如可向旅遊網站購買旅遊票券，或是線上購票看表演、展覽、比賽等。
